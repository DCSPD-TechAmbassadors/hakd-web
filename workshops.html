8-bit Chiptune Game Music

Mr. Bennington

Did you know the whole original Super Mario Bros soundtrack was only 17kb? That’s 0.5% of a single mp3 file! It also only used 4 audio channels, compared to today’s songs which often use hundreds!

Discover the origins and evolution of video game music, while learning about digital sound creation and programming. You will also create your own 8-bit music that has inspired such recent soundtracks as Undertale and Shovel Knight.

No music composition experience is required, but some musical knowledge is recommended. No software will be installed, as we will use web-based tools that will work on any browser and OS.

Object Oriented Programing (OOP) with Python

Coll

Come learn about Object Oriented Programming (OOP), a programming style which offers a range of alternative benefits compared to procedural programming, which is taught in most school curriculums. We'll learn about objects, methods, inheritance and other OOP techniques. By the end of the workshop, you'll have created a Python Turtle-based object (Turtle is a drawing library), which will use inheritance and customized methods. You'll also be able to use Object Oriented Programming to solve various problems, and you'll be able to identify when it's best to use procedural or OOP techniques. A good level of Python skills is required, and you should be comfortable with procedures and functions – if you’re unsure, come talk to me before signing up.

Requirements: A good level of Python skills. Any Python IDE installed
