Introduction to Arduino

Mr. Whitaker

In this workshop be prepared to roll up your sleeves and get up close and personal with an Arduino board! You will build and program a small control system that uses sensors, buzzers, LEDs and more. In the process you will learn about analogue and digital I/O (input and output) as well as Arduino programming fundamentals. Please download and install the Arduino IDE before this workshop if possible.

This workshop is presented by Mr. Whitaker, Director of Ed Tech at DCSPD and an inspiration to his IB Computer Science class!

Recommended for people who are willing to learn a new skill with hardware.
Download Arduino: https://www.arduino.cc/en/software

Marvelous Minecraft

Mr. McFadden

Introduction to Microsoft Makecode in Minecraft Education Edition.

Workshop participants will be introduced to programming in Minecraft Education Edition using Microsoft Makecode. Participates can choose to program in Block (Scratch like), Python or JavaScript, depending on preference. No prior programming experience is required to get started although there is a range of opportunities and challenges for everyone to get involved.

It would be helpful if prospective participants could pre-install Minecraft Education Edition from https://education.minecraft.net/en-us/get-started/download. Please note that this is not the same as the standard edition Minecraft.

Online databases with Airtable

Hervé

Have you ever wanted to store images and other data online as part of your retaliation against the Galactic Empire? Well, maybe just the first part… But if you have wanted to do that, then look no further, this workshop is for you!

In this workshop you will learn the essentials of accessing an online Airtable database via a Python API. Airtable is a free online database service that offers an intuitive spreadsheet like interface as well as easy to use APIs in a few handy programming languages. The skills taught in this workshop may be useful for any of your projects that require off-site data storage.

This workshop is brought to you by Hervé, a Y13 tech enthusiast, aspiring scientist and the previous vice-president of the Tech Ambassadors.

Previous basic programming experience expected. Please have Python 3 installed on your computer in advance if you are going to attend.

Basis of 3D modeling with Blender

Daniel & Ben

Have you ever wondered about how your favorite item within a video game is made? Well, if you did, this workshop is just for you. In this workshop we will teach you how to do the basics of sculpting and animating 3d objects with a free and open-source software called Blender.

In this workshop you will learn about how to use the basic features of Blender to help you with your project, such as keyframe animations, physics simulations, modifiers, and more!

Hosted by: Ben and Daniel: year 10 computer science students.

Dashing Robots

Dr. Yankelevich

Discover the wonderful world of the Wonder Workshop with Dash! Dash is a playful robot designed for easy access to the world of coding and robotics. Dash uses Bluetooth, 3 microphones, speakers, 3 processors and sensor fusion, proximity sensors, motors, and LED lights to create a super fun user experience. Come join the fun, all ability levels welcome as Dash uses simple Blockly (space is limited)!

MicroBits

Chase

Fn this workshop you will learn about the micro:bit which is a handheld, programmable micro-computer that can be used for all sorts of cool creations, from robots to musical instruments – the possibilities are endless! You will program the LEDs as an output based on various inputs. We will be working with block code but you can also use Python through the online micro:bit coding interface.

This workshop is presented by Chase Yiu from Y8. This workshop is most suitable for students with no or little experience in hardware and software.
